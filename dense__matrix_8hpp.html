Detailed Description

The Dense Matrix Class. A template class which is used to handle two-dimensional arrays within the program. In the documentation, we will used the letter to refer to an arbitrary matrix, and to refer to the matrix element at row i and column j. The memory is allocated in an array following row major format. The rows and columns are assume to follow a zero-base index convenction.
