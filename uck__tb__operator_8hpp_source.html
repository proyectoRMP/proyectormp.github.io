This constructor creates a tight-binding operator for a system with dimension OpDim. If ommited, the constructor assumes OpDim=1. The constructor allocates some memory for the initialization, but does not allocate the memory for the tight-binding elements of the operator. The tight-binding elements should initialized later with a call to readOperator(), which will also allocate the necessary memory.
