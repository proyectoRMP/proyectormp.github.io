This Function perform the linear transformation . Here, |x and |y are complex state vectors store in a continous array, and a and b are real scalars. To improve efficiency, this function does not check for out of bound errors, therefore should be check prior calling this function.

Parameters
a.A scalar paramater that will scale the input vector
X.A pointer to a complex array where the input vector is located.
b.A scalar paramater that will scale the ouput vector
Y.A pointer to a complex array where the output vector is located.
Returns
status The status is a datatype defined in uck_tb_operator_error.hpp, returning TightBinding::status::SUCCESS if successful or TightBinding::status::FAILURE otherwise.
