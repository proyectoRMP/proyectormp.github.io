Detailed Description

This header defines the parsing functions of the code. They are mostly used to extract parameters from from configuration and data files, and transform these into data structure such as datatypes, vectors, arrays, and matrices.

Look for a field in config or data file and extract its value. Here we define field everything that is before the equal sign in a config file. The field should be alphanumeric, without spaces. The function determines if the value is suitable and transform it into a valid datatype.

Parameters
filenameString with the name of the file.
fieldString with the name of the field.
valueReference to a valid datatype where the value will be stored
Returns
parser::error::SUCCESS if correct and parser::error::FAILED if not accesible
//EXAMPLE
#include <fstream>
#include <cstdlib>
#include <string>
#include "parser.hpp"
int main()
{
std::string filename( "NotAccesibleFile" );
if( !parser::FileReadable(filename) )
std::cerr<<"The file is not accesible"<<std::endl;
filename= "config" ;
if( parser::FileReadable(filename) )
{
std::cout<<"The file is accesible"<<std::endl;
double value;
std::string field="myValue";
int status = parser::GetValue(filename,field,value);
std::cout<<std::endl<<"myValue="<<value<<std::endl;
if(status)
std::cout<<"Extraction Sucessfull"<<std::endl;
std::string str;
field="myString";
status = parser::GetValue(filename,field,str);
std::cout<<std::endl<<str<<std::endl;
if(status)
std::cout<<"Extraction Sucessfull"<<std::endl;
}
return 0;}
