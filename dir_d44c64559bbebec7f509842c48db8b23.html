file  uck_tb_operator.hpp [code]
 This Class is the responsible for handling the tight-binding operator. The tight-binding operator consists in a list of hoppings t_ij between orbital orbi and orbj, which can belong to two different unit cells defined with indexes (i0,i1,i2) and (j0,j1,j2)=(i0+dij0,i1+dij1,i2+dij2).

The tight-binding operator is initialize by reading the operator from a .KOP file, which should be placed in the operator directory. This file will contain the number of orbitals per cell, The total number of hoppings, the lattice constant and lattice vector and the list of t_ij, orbi, orbj, (dij0,dij1,dij2).

The class allows for creating a dense or sparse matrix with different boundary conditions, including twisted boundary conditions (k-dependent).
