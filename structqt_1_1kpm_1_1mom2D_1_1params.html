status qt::kpm::mom2D::params::ReadCFGFile ( const std::string  config_filename)

The method ReadCFGFile read the KPM parameters from the config file. These parameters will vary depending on the kind of calculations. The following parameters MUST BE in the config file:

  • label
  • MaxMoments
  • EnergyShift
  • EnergyScale
  • Optionally a suffix. After reading this, it also initialize mom1D as a vector of size the maximum number of moments

    config_filename The name of the configuration file
    Returns
    kpm::SUCCESS if all the parameters where properly initilize. kpm::FAILURE if not.
