qt::UnitCellK_TBOp::UnitCellK_TBOp ( const qt::integer  OpDim = 1)
inline

This constructor allows to initialize the tight-binding operator for systems with dimension OpDim. Optionally, this variable can be omitted and the constructor will assume OpDim=1; This constructor will allocates some memory for the initialization of some of the internal arrays.

qt::UnitCellK_TBOp::UnitCellK_TBOp ( std::string  filename)

This constructor allows to initialize the tight-binding operator for systems defined by a file. This constructor will allocates the necessary memory for hosting the full operator, except for the dense and sparse matrices which need to be initialize with specific methods.
